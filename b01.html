다음은 티스토리에서 사용할 수 있는 HTML 표 서식입니다. 이미지와 함께 각 MBTI 유형을 클릭할 수 있도록 링크가 설정되어 있으며, 사용자는 각 이미지에 대한 링크와 설명을 직접 수정하시면 됩니다. 표는 요청하신 대로 **5행 x 5열**이며, 두 번째 행부터 네 번째 행까지는 이미지와 텍스트가 있는 구조입니다. ```html
MBTI 가이드
ISTJ
ISFJ
INFJ
INTJ
ISTP
ISFP
INFP
INTP
ESTP
ESFP
ENFP
ENTP
ESTJ
ESFJ
ENFJ
ENTJ
``` ### 사용 방법 - `"이미지링크"` 부분에 각 MBTI 유형에 맞는 이미지 주소를 입력하시면 됩니다. - `` 내부의 `#` 대신 각 MBTI별 링크 주소를 넣으시면 클릭 시 해당 페이지로 이동됩니다. 필요하시면 제가 깃허브 이미지 링크를 넣은 버전도 도와드릴 수 있습니다. 이미지 URL만 주시면 바로 적용해드릴게요.
